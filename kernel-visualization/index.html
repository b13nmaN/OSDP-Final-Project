File System Operation

When a process needs to read or write files, it must request the kernel's help. The kernel:

  1. Checks file permissions
  2. Manages file system cache
  3. Handles actual disk I/O

Memory Management

When a process needs more memory, the kernel:

  1. Checks available memory
  2. Allocates new memory pages
  3. Updates page tables

Network Communication

When a process needs to send/receive data over network, the kernel:

  1. Manages network protocols
  2. Handles packet routing
  3. Controls network interfaces
